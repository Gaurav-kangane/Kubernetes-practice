2. create statefulset, pv, pvc, secret, clusterip and backend files and apply all --> kubectl apply -f .

3. kubectl get pods --> 3 pods will be create 

4. kubectl get svc --> check headless service --> kubectl describe mysql service

5. kubectl get pv , kubectl get pvc

6. log in the dns-client pod --> kubectl exec -it dns-client -- sh

7. You will be logged in to the container run this command  --> nslookup <service-name>

8. Test both servcies headless and clusterip and observer how traffic is routing
